Core PostgreSQL Tables

Contract Rules

  • execution_traces is the replay source of truth.
  • confidence_score stored in [0.0,1.0], UI shows percent.
  • Run dedupe keyed by fingerprint, not run_id.
  • results.json required for all terminal runs.
